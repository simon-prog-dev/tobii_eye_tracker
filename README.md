# tobii_eye_tracker
Provide solution to know what user is watching. Using Tobbi Glasses pro 2 and tiny_yoloV4.
A simple kivy interface is used.

installation necessary:   kivy,cv2,tobiiglassesctrl,numpy

#to run it

import tobbi_app

tobbi_app.TobiiApp().run()
